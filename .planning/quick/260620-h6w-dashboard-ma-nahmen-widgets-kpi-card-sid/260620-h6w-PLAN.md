# Plan: Dashboard Maßnahmen-Widgets

Zwei optionale Maßnahmen-Widgets einführen, beide per Default DEAKTIVIERT.
Nutzer aktivieren sie über "Kacheln anpassen".

## Task 1: Dashboard.tsx – KPI-Kacheln anpassen
- `openMeasures` + neue `overdueMeasures` aus DEFAULT_KPI_IDS entfernen
- `sidebarMeasuresBadge` zu ALL_KPI_IDS hinzufügen (virtuell, kein Card)
- `overdueMeasures` zu kpiConfig + kpiLabels hinzufügen
- `sidebarMeasuresBadge` nur in kpiLabels (kein Card-Rendering)
- Kacheln-Popover: Trennlinie zwischen Dashboard-Kacheln und Sidebar

## Task 2: MainLayout.tsx – Sidebar-Badge

- State: showMeasuresBadge + openMeasuresCount
